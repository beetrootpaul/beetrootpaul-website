# beetrootpaul-website

Codebase of the beetrootpaul.com website

## Development

```shell
nvm use
npm install
npm run dev -- --open
```

```shell
nvm use
npm install
npx playwright install
npm test
```

## Static build

```shell
npm run build
npm run preview -- --open
```
